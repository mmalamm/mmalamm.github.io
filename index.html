My Story

I'm Muhammad Alam, a software engineer from New York City who loves problem solving and fine dining. I write applications in Rails, Node.js, express.js, SQL/noSQL, React/Redux, D3 and more. I have been programming all my life, but my first academic experience in programming was in a bioinformatics lab in Stony Brook University, where we used R and Python to analyze genetic data from NCBI. I quickly became hooked on the power and flexibility of coding when it comes to problem solving. After college, while serving in the Army and teaching/tutoring for the Princeton Review, I taught myself full stack web engineering in JavaScript, and then attended App Academy, a top intensive coding bootcamp in New York City, where we focused on the principles of Object Oriented Programming and full stack web engineering using React/Redux on Rails/postreSQL. I am currently volunteering part-time as a web developer with the United Nation's Project to alleviate Poverty, which is a part of the UN's 2017 SDG's.
